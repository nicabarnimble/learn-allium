# Exercises

All completed `.allium` files here are validated against `allium 3.5.0`
(`check` + `analyse`). Starters are intentionally incomplete or flawed;
solutions are clean.

| Exercise | Used in | What it trains |
|----------|---------|----------------|
| [01 — First distill](01-first-distill.md) | Session 1 homework | Backward spec-from-code |
| [02 — Access grant lifecycle](02-access-grant-lifecycle/) | Session 2 exercise | End-to-end Allium-driven feature |
| [03 — Spec refactor](03-spec-refactor/) | Session 3 exercise | Anti-pattern hunting, analyse findings |
| [Capstone](capstone.md) | Sessions 5–6 | Full loop on real code |

Working rule for every exercise: **no code before the spec is green**, and
every claim of "done" is a clean `allium check` + `allium analyse` run.
