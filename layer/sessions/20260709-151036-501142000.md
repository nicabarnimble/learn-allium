A controlled `openai-codex/gpt-5.4-mini` + `minimal` request answered the test exercise in about 12 seconds.

- Latency feedback led to separating model quality from thinking effort. Users can select any model available from `pi --list-models`, while low/minimal thinking is sufficient for small teaching explanations.
- Liveness feedback led to visible progress rather than leaving the terminal apparently frozen during a one-shot model request.
